# 팀명 : 배부르조 운동하조
## 주제 : 개인 맞춤형 운동 추천 서비스
+ 프로잭트 설명 : 개인 성향에 맞는 운동을 추천해주고 주변 체육관,PT트레이너,추천 영상을 추천해주는 웹서비스
## 1.주요기능
+ 설문조사기능
+ 주변 PT트레이너/체육관 추천기능
+ 영상 추천 기능
+ 캘린더 기능
